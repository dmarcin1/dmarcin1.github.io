1. 创建新仓库

点击“Repositories”标签，然后点击右上角的“New”，新建一个仓库，其中仓库名称为 xxx.github.io，xxx 是 GitHub 的账号名。
这个仓库名称必须严格按照这个格式创建，否则后面 Page 页无法访问。


2. 下载安装 Git 和 Node.js

直接去官网下载安装 Git 和 Node.js，这里不详细写过程。
安装完成后，打开 cmd 命令窗口，执行以下命令，查看是否安装成功：

node -v
npm -v

3. 生成 SSH Keys

打开 cmd 命令窗口，执行以下命令：

git config --global user.name "注册时用户名"
git config --global user.email "注册时邮箱"

上述命令中，填写你的 GitHub 用户名和邮箱。
接着执行以下命令生成 SSH Key：

ssh-keygen -t rsa -C "你的Github邮箱地址"

生成过程中，一直点击回车即可完成。

找到刚才生成的 SSH 文件保存路径，默认是在 C盘 用户目录下的 .ssh 文件夹中。
找到 .pub 后缀的文件，用记事本打开，复制其中的文本内容。

然后，打开 GitHub 页面，点击右上角的个人头像，进入设置页面：

在设置页面中，找到 SSH and GPG keys，然后点击 “New SSH key”，将之前复制的文本粘贴进去。
