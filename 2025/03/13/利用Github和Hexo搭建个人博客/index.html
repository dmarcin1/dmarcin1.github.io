1、创建新仓库

点击“Repositories”标签，然后点击右上角的”New”，新建一个仓库，其中仓库名称为xxx.github.io,xxx是github的账号名，==这个仓库名称必须严格按照这个格式创建，否则后面Page 页无法访问==
![[Pasted image 20250313102442.png]]
![[Pasted image 20250313102329.png]]

2、下载安装git 和 node.js ，直接去官网下载安装，就不详细写过程了。

安装玩后打开cmd命令窗口，执行node -v  和npm -v 命令，查看是否安装成功。

![[Pasted image 20250313103137.png]]

3、生成ssh-keys

打开cmd命令窗口执行下面命令：
git config --global user.name "注册时用户名"
git config --global user.email "注册时邮箱"
github的用户名和邮箱
 ssh-keygen -t rsa -C "你的Github邮箱地址"，然后一直点击回车就好啦

![[Pasted image 20250313103824.png]]
找到刚才ssh文件保存的路径，默认是在c盘用户下有个.ssh文件夹，找到里面的.pub后缀的文件，用记事本打开，复制里面的文本。
![[Pasted image 20250313103645.png]]

打开github的页面，点击右上角的个人头像，进入设置页面
![[Pasted image 20250313104142.png]]
找到SSH and GPG keys ，然后点击 “New SSH key”,将上面复制的文本粘贴进去。
![[Pasted image 20250313104219.png]]
